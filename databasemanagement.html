This page is about database management system:

Data

Data, in the context of databases, refers to all the single items that are stored in a database, either individually or as a set. Data in a database is primarily stored in database tables, which are organized into columns that dictate the data types stored therein

Information

Information in a database is a collection of structured data that is organized and stored electronically in a computer system. Databases are used to store large amounts of data in one place so that organizations can easily access, manage, update, and retrieve it.

Metadata

Metadata in a database is information that describes the structure, contents, and context of the database. It's also known as "data about data" and can include information such as: Table and column names and types Table relationships Data constraints Indexes Views Other database objects Data source and history.

Database

A database is a structured collection of data that is usually stored electronically in a computer system. It can contain any type of data, including words, numbers, images, videos, and files. Databases are used to store and manage large amounts of data, and they can be used to support a wide range of activities, including data storage, data analysis, and data management

Tuples

A tuple is a fundamental unit of data that represents a single row or record in a database table. Tuples are also known as rows or records.

Table

Tables are fundamental objects in a database that store and organize related data in a row-and-column format, similar to a spreadsheet. Each row represents a record, and each column represents a field within that record. Tables can also have properties that control their appearance and behavior.

Fields

A field in a database is a column or attribute that contains a specific type of data and defines the values each record can have in a table. Fields can be used to store information such as names, addresses, or other data.

Domain

A domain in a database is a set of values that can be assigned to a column or attribute, and it establishes the data type and range of values that can be stored in that field. Domains can be built-in types, like strings or integers, or custom types that define constraints on the data.

Constraints

Database constraints are properties that restrict data in various ways.

Keys

It is used to uniquely identify any record or row of data from the table. It is also used to establish and identify relationships between tables.
